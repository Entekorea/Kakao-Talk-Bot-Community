[주로 일어나는 오류]
· Compile Error(봇 이름): org.mozilla.javascript.EvaluationException: missing } after function body (봇 이름#n)
중괄호가 없을 시 나타나는 오류입니다. n번째 줄 부분을 한번 검토해보시고, 중괄호를 꼭 닫아주세요.

· ReferenceError: ~ is not defined.
~가 정의되지 않아(사용자마다 ~에 들어가는 값은 다릅니다) 발생하는 오류입니다. 변수 ~를 정의해주시거나, ""로 감싸주시면 해결됩니다.

· Compile Error(스크립트 이름): 'try' without 'catch' or 'finally' (스크립트 이름#n)
try-catch 문에서 catch 혹은 finally 가 빠져서 발생하는 오류입니다. 만약 스크립트에 catch(or finally) 문이 있는데도 이런 오류가 발생한다면 중괄호의 위치가 잘못된 것이니, 중괄호들을 다시 한 번 확인해주시기 바랍니다

· SyntaxError.
구문 오류로서 스크립트 내에 잘못된 철자와 컴퓨터가 인식하기 어려운 문자, 자바스크립트 문법에 어긋난 경우에 발생합니다. 보통 세미콜론을 붙여야 하는 곳에 세미콜론을 붙이지 않거나 괄호 등을 열고 닫지 않으면 발생하기 때문에 Error가 발생한 줄의 구문을 다시 한 번 확인해 주시기 바랍니다.

· Runtime Error(스크립트 이름): TypeError: Cannot find function trim in object ~~. (스크립트 이름#n) at 스크립트이름:n (response)
TypeError 는 자료형에 맞지 않는 프로토타입 메서드 혹은 메서드를 사용할 때 발생하는 오류입니다. 해당 자료형에 알맞는 메서드를 사용해 주시길 바랍니다.

· Runtime Error(스크립트 이름): Error: Module ~~ not found. (스크립트 이름#n) at 스크립트이름:n​

~~모듈이 없는경우 발생하는 오류입니다. sdcard/katalkbot/Bots/스크립트이름/modules 또는 sdcard/katalkbot/Bots/global_moduels 에 해당 모듈을 넣어주세요.
